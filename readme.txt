== Description ==

Use the system "twitter card" & "open graph" proposed by twitter wordrpess on your site.

Twitter card :

* Add card
* Add image
* Add site
* Add creator

Open Graph :

* Add URL
* Add title
* Add description
* Add site_name

More informations [twitter card](https://dev.twitter.com/docs/cards) & [content tagged with twitter card](https://dev.twitter.com/tags/twitter-cards)

PS: question, [eewee.fr](http://www.eewee.fr).

* See the [Changelog](http://wordpress.org/extend/plugins/eewee-pinterest/changelog/) for what's new.

== Installation ==

1. Download the plugin Zip archive.
1. Upload plugin folder to your '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
